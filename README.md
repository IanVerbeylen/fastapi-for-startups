# FastAPI for Startups

Production-grade FastAPI development for building real products. This repo accompanies the [YouTube series](https://www.youtube.com/@startup_engineering) where I build a valuation platform from scratch.

## What You'll Learn

- Clean architecture and project structure
- Database design with PostgreSQL, Async SQLAlchemy and Alembic
- Authentication and security best practices
- Comprehensive testing strategies with Pytest
- Docker and containerization
- CI/CD pipelines
- Background tasks
- Production deployment
- Real architectural decisions for a SaaS product

## Structure

Each episode has its own branch with:
- Complete working code
- README explaining key concepts
- Links to the corresponding video

## Tech Stack

- FastAPI
- PostgreSQL
- Docker
- SQLAlchemy
- Alembic (migrations)
- Pytest (testing)
- GitHub Actions (CI/CD)
- Celery & Redis
- uv
- ruff
- pre-commit
- and more

## Following Along

Start with episode 01 and work through sequentially. Each episode builds on the previous one.
